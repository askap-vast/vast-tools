    Changelog¶

    All notable changes to this project will be documented in this file.

    The format is based on Keep a Changelog, with an added List of PRs section and links to the relevant PRs on the individal updates. This project adheres to Semantic Versioning.

    3.2.0 (2024-05-08)¶

    Added¶

    • Add epochs 73 and 74 #598
    • Add epochs 71 and 72 #583
    • Add epoch 69 #576
    • Add epoch 68 #575
    • Add epoch 67 #573
    • Add epoch 66 #569
    • Add epochs 64 and 65 #568
    • Support for Python 3.10 #334
    • Add access to epoch 63 #563

    Changed¶

    • Calculate cutout normalisation each time, rather than storing a normalisation with no easy way to change it #581
    • Renamed "force" variable to "force_cutout_fetch" for clarity #581
    • Allow users to specify the size of a single cutout in show_all_png_cutouts, rather than forcing them to specify the total figure size #581
    • Updated vasttools.source.Source logger to include the source name #581
    • Minor changes to docstring formatting throughout based on updated mkdocs versions #334
    • Minor changes for matplotlib 3.7: add angle kwarg to Ellipse and change matplotlib.pyplot.cm.get_cmap to matplotlib.colormaps.get_cmap #334
    • Refreshed dependencies - major changes are python 3.10, mkdocs (and related packages), astropy v5 and matplotlib v3.7 #334
    • Update emoji names to reflect latest fontawesome naming scheme#334
    • Fixed minor typos in docs and docstrings #334
    • Updated old contact information #334
    • Updated github actions to latest versions #562
    • Updated 2023 workshop notebook - fixed Query options and general cleanup #558

    Fixed¶

    • Updated open_fits to correctly handle NaN padding in CompImageHDUs #597
    • Updated ubuntu-20.04 -> ubuntu-24.04 in github workflows #592
    • Correctly calculate plot_lightcurve y-axis range for multi-frequency upper limits #567
    • Fixed formatting of new epoch addition git example #568
    • Directly compare stmoc times, avoiding conversion to JD, and replace equality requirement with isclose #334

    Removed¶

    • Removed lightgallery functionality from docs #334
